# 📊 Sales & Product Performance Dashboard (Power BI)

## 🚀 Project Overview
This project showcases an interactive **Sales & Product Performance Dashboard** built using **Power BI**.  
The goal is to transform raw retail data into actionable business insights through data modeling, DAX, and modern dashboard design.

---

## 🎯 Objectives
- Analyze overall sales performance
- Identify top-performing products
- Evaluate profitability across categories
- Explore regional sales distribution
- Understand the relationship between sales and profit

---

## 📸 Dashboard Preview

### 🔹 Sales Overview
- Total Sales, Profit, Orders, Return Rate
- Sales trend over time
- Sales by category
- Profit by sub-category
- Sales by region

### 🔹 Product Performance Analysis
- Top 10 products by sales
- Top 10 products by quantity
- Profit by sub-category
- Sales vs Profit (scatter analysis)

---

## 🛠 Tools & Technologies
- **Power BI**
- **DAX (Data Analysis Expressions)**
- **Data Modeling**
- **Excel (Sample Superstore Dataset)**

---

## 📊 Key Features
- Interactive slicers (Category, Sub-Category, Product, Date)
- KPI cards for quick insights
- Conditional formatting for profit/loss analysis
- Dynamic filtering across all visuals
- Clean and modern UI design

---

## 💡 Key Insights
- Most sub-categories are profitable, but a few generate consistent losses
- Certain products drive high revenue but lower profit margins
- Sales performance varies significantly across regions
- Strong correlation observed between sales and profit for top products

---

## 📂 Dataset
The dataset used in this project is the **Sample Superstore dataset**, commonly used for learning and portfolio projects.
